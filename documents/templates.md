# Document Templates

Our team is working on building document templates for Word that when exported using Word's built-in Save as PDF option will result in a fully compliant accessible document. These templates can be used as-is or as a reference for building your own templates according to the above guide.

We will add to these templates over time. If you have a suggestion for a type of document we could add a template for, let us know!
